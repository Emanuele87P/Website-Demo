Scratch

Intro

Scratch is a programming language that allows you to create animations and games using block code.
Scratch is used by users of all ages.
With scratch people create projects in house, school beacuse it is free.
Scratch designs are simple are simple to use,
children and adults can use it.
The disadvantage is that its visual interface is different from other computer programs that are
created with a text-based system. Scratch does not teach coding languages.

History

Scratch was created by MIT Media Lab's Lifelong Kindergarten group and consulting firm, Playful
Invention Company, in 2003.
Scratch's name comes from the technique used by DJs. This name, of
course, is by no means accidental and accurately describes the basic idea of Scratch, namely that of
combining images,
sounds and texts together to create multimedia content.

Charateristics

Scratch as we said before uses a block code,
its main screen is divided into 4 distinct areas and we have:

  • The stage is the large white rectangle that is placed in the upper left corner and will be the "stage";

  • The Sprite (the characters) is a smaller window located below the stage;

  • The Block List is located in the center of the screen and is a column that contains all the programming

    blocks. Each type of block has a specific function and its own color, all this to distinguish it;

  • The Script Area is the area where we insert the blocks to create the scripts.
