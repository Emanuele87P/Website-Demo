Intro paragraph

Java is a programming language famous for its ability to compile to platform-independent bytecode.
It powers enterprise web apps, big data pipelines, and android mobile apps.

History paragraph

Designed by James Gosling in 1990s in Sun Microsystem;
One of its first demonstrations was the STAR7 PDA (Personal Digital Assistant) which gave birth to the Java Mascot DUKE;
Today it’s one of the most popular programming languages, examples it powers:

  • Enterprise Web Apps //with Spring Framework;
  • Big data pipelines //with Hadoop;
  • Mobile apps //with Android;
  • Nasa’s maestro //the Mars Rover.
Charateristics paragraph

High usability scheme, independent from the operating system:

  • High level code >> (compiler) >> file.class >> Interpreter.

Instead of compiling two machine code, it compiles two-byte code that can run on every operating
system without recompiling by a Java Virtual Machine (both compiler and interpreter);
The computer just needs to have a Java runtime environment installed to run a code;
for developers:

  • WRITE ONCE, RUN ANYWHERE.
