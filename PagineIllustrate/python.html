Sintax paragraph

Python, compared to other programs, commands are written on a single line and does not require
special termination characters (such as in Java and C++ we use the (;) to terminate the command), to
write more commands on the same line they can be divided using the (;). If we want to identify
variables, functions, classes and modules we initialize them with a letter or a (_). For Python it is a case
sensitive programming language, that is, uppercase and lowercase characters are considered two
different variables (for example Name and Name are different).

  • Class names begin with an uppercase letter, all other identifiers begin with a lowercase letter;
  • An identifier starting with an underscore indicates that it is a private identifier;
  • An identifier starting with two underscores indicates that it is a strongly private identifier;
  • If the identifier begins and ends with two underscores then it is a special name defined in the language.

As we can see between the two programs the differences are few:

  • To initialize a function in C++ we use for example “int” instead in Python we use the “def”;
  • In Python we do not use curly brackets but the ":" and the rounds are not necessarily usable.
    In order for the program to accept that the command is written on several lines, it is necessary to use the "\" except in cases where parentheses are used.

To comment in Python:

  • Comments should first be written on the same level of identification as the code you are
    commenting on and it is best to avoid comments on the same line of code.
  • You can use “#” to comment on the same line or the (“ " ”) to comment on multiple lines.
